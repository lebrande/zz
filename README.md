# How to run?
`npm start` or `yarn start`

# Tools I used
- I use *Parcel* as assets bundler and development server.
- All front-end scripts written in *TypeScript*.
- *styled-components* - for CSS styles.
- *SVG* - for icons.
- *CSS* - rendering simple shapes.
